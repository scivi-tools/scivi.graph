Основные действия:
  • LMB – выделить вершину
  • Drag – переместить вершину
  • Shift + LMB – выделить несколько вершин
  • RMB на кольцевой шкале – получить контекстное меню фильтрации
  • Shift + LMB на секторе кольцевой шкалы – показать только дуги, один из концов которых находится в этом секторе
  • Alt + LMB на секторе кольцевой шкалы – показать только дуги, оба конца которых находятся в этом секторе
